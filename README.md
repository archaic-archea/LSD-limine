## LSD-limine
LSD is a WIP operating system, it currently supports a simple userspace, and a few syscalls which can be accessed through `lsd_lib`

## Running
```
$ git clone https://github.com/archaic-archea/LSD-limine
$ cd LSD-limine
$ cargo xtask run
```
## Building
```
$ git clone https://github.com/archaic-archea/LSD-limine
$ cd LSD-limine
$ cargo xtask build
```
