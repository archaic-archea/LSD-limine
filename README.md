## RISC-V Barebones
This is a small repository to act as a barebones start for RISC-V OSDev

## Running
```
$ git clone https://github.com/archaic-archea/risc-v-bare
$ cd risc-v-bare
$ cargo xtask run
```
## Building
```
$ git clone https://github.com/archaic-archea/risc-v-bare
$ cd risc-v-bare
$ cargo xtask build
```
